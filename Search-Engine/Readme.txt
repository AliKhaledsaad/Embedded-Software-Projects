this project is about a search system user can enter any numbers for a list of 16 number 
each number user enter must press "+" to isolated from other numbers when finished enter "ON"
to start entering number you look for then press "=" 
-if the number in the list it will be displayed at the 7segment and in a binary version on the leds which represent 1-2-4-8 from up to down(if number is 0 leds will be off)
